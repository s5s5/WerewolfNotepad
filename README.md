# 🐺 狼人杀记录器

一款简洁高效的狼人杀游戏记录工具，帮助你在游戏中快速记录每位玩家的发言要点。

## ✨ 功能特点

- 🎯 **快捷记录** - 预设常用狼人杀术语按钮，一键输入
- 👥 **灵活人数** - 支持 6-18 人局，自由配置
- 💾 **自动保存** - 数据本地存储，刷新不丢失
- 📱 **响应式设计** - 支持 PC 和移动端
- ⌨️ **快捷键支持** - 高效操作，解放双手
- 📤 **导出功能** - 一键导出记录为文本文件

## 🎮 使用方法

### 基本操作

1. **设置人数**：在右上角输入玩家人数，点击齿轮按钮确认
2. **选择玩家**：从下拉框选择当前发言的玩家
3. **记录发言**：
   - 点击快捷按钮快速输入关键词
   - 或直接在文本框输入内容
4. **完成记录**：点击「记录」按钮或按 `Ctrl+Enter`
5. **切换玩家**：点击「上一位/下一位」或按方向键

### 快捷按钮说明

| 分类 | 按钮 | 用途 |
|------|------|------|
| 身份 | 预言家、女巫、猎人、守卫、民、神 | 玩家声明身份 |
| 验人 | 金水、查杀、银水、警徽流 | 预言家/女巫验人结果 |
| 站边 | 站边、怼、狼坑、绑定、出局 | 玩家之间的关系判断 |
| 疑点 | 逻辑问题、状态问题、言票不一、划水 | 标记可疑行为 |
| 技能 | 救、毒、开枪、守护 | 神职技能使用 |
| 玩家 | 1-12（根据人数） | 快速输入玩家号码 |

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + Enter` | 记录完成并切换下一位 |
| `←` / `→` | 切换上一位/下一位玩家 |
| `Esc` | 清空输入框 |
| `Ctrl/Cmd + S` | 手动保存 |
